SaaS Deployment Updates
On December 6th, Learn SaaS v3500.5.0 was released to production environments. This release brings highly anticipated features and enhancements to the Ultra experience, including:
Ultra Course View
  • Learning modules now available in Ultra courses to enable students to work on content in a specified sequence without needing to close and open content items
  • Initial availability of the Kaltura integration within Ultra courses
  • Expanded user privileges with respect to gradebook access and control (also known as content lockdown)
  • Course groups feature allows instructors to create groups and group sets at the course level, outside of assessments
  • Ability to batch edit due dates in course content, building on a previous feature to batch edit visibility of course content
On January 3rd, Learn SaaS v3500.7.0 was released to production environments. This release brings exciting new features and enhancements, including:
Original & Ultra Course Views
  • Newly redesigned Originality Report available for SafeAssign
  • Caliper 1.1 update for Blackboard Learn Telemetry Service
Ultra Course View
  • Learning modules are preserved in course copy, export, import, and restore operations when converting from Original to Ultra
  • Expanded gradebook search capabilities with the ability to search the submission list
  • Updated the name of the "Item Analysis" feature to "Question Analysis" to increase awareness and adoption of this important report, also added the ability to download the report
View all the details in the release notes here.
