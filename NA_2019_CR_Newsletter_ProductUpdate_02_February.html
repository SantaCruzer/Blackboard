SaaS Deployment Updates
On January 31st, Learn SaaS v3500.9.0 was released to production environments. This release brings highly anticipated features and enhancements, including:
Original & Ultra Experience
  • Instructors can now host a Collaborate session without having to spend time tracking attendance manually in Learn; Collaborate automatically tracks if a student is present, late, or absent from a session and sends that information directly to the Attendance page in Learn, deepening the seamless integration between our products
  • Expanded user privileges with respect to course calendar control (also known as content lockdown)
Ultra Experience
  • Ability to select a theme color to reflect institutional branding
  • Users can now select which push notifications they would like to receive on their mobile devices
Ultra Course View
  • Expanded gradebook search capabilities with the ability to search columns and categories
  • Respondus Lockdown Browser, a custom browser that locks down the testing environment, is now seamlessly integrated within Learn and built into the assessment settings
  • Incorporated additional SCORM functionality to include advanced settings functionality and reporting analytics for instructors
View all the details in the release notes here.
 
Blackboard Collaborate
Collaborate Attendance Integration with Blackboard Learn
Blackboard Learn instructors can now host a Collaborate session without tracking attendance manually! Collaborate automatically tracks if a student is present, late, or absent from a session and sends that information directly to the Attendance page in the Learn course.
Metric, Recording, and Attendance reports available from Collaborate Scheduler
You no longer need to submit a support ticket for data about usage statistics. With this release, managers can view their institution Metric, Recording, and Attendance reports from the Collaborate Scheduler.
View all the details in the release notes here.
 
Blackboard Ally
Alternative Formats and Content Feedback for Web Community Manager
The Ally Alternative Formats are now available for the Web Community Manager integration along with the Ally Content Creator feedback. Both new features should now be available for all Web Community Manager schools that have Ally integrated.
Previews and Highlights
The instructor feedback now shows in-browser previews for PDFs, Word documents and PowerPoint documents. These previews can be used to identify where in the document specific accessibility issues can be found.
