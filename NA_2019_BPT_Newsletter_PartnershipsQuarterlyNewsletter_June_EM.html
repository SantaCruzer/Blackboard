Important Technical Updates
 
 
IMPORTANT REMINDER: Upcoming Changes to Blackboard Learn May Break Building Blocks (B2s)
As you may know, Blackboard Learn has transitioned to adopting the latest version of Open JDK as it is released. We want to remind you that will be updating Blackboard Learn SaaS with JDK 11 to incorporate the fixes made by Oracle to the Java support model. We plan to push the changes to client-test SaaS systems in mid-July. Your current Building Block may be using libraries or Java features that are not compatible with OpenJDK 11, causing it to fail.
It is critical to test your Building Block(s) on the OpenJDK 11 build of Blackboard Learn. To learn important information about this upcoming release and how to participate in a Technical Preview, please review this article and FAQ. Sign up for the Learn SaaS Continuous Delivery Java 11 Technical Preview Program here.
